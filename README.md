# Vingle Design System

vingle 프로젝트에서 사용중인 design system 입니다.

## How To Install

먼저, 의존성을 설치해줍니다.
```
npm install vingle-design-system
```

사용할 컴포넌트 최상단에 제공하는 Provider를 감싸주세요.

```tsx
import { StyleProvider } from "vingle-design-system";

// ....
<StyleProvider>
{/* your component place in here */}
</StyleProvider>


```
