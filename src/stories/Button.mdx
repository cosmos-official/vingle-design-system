# Button

Button 컴포넌트는 사용자의 상호작용을 유도하는 버튼을 제공합니다.

## 디자인

<Canvas of={ButtonStories.PrimaryBlackButton} />
<Canvas of={ButtonStories.PrimaryPinkButton} />
<Canvas of={ButtonStories.GrayButton} />

## props

- `onClick: () => void`: 버튼 클릭 시 작동할 이벤트 핸들러를 정의합니다.
- `children: ReactNode` : 버튼에 나타낼 내용을 설정합니다.
- `variant?: 'primaryBlack' | 'primaryPink' | 'gray' | 'none`: 버튼의 스타일을 설정합니다. 기본값은 'none' 입니다.
- `size?: 'xm' | 'sm' | 'm' | 'l' | 'xl' `: 버튼의 크기를 설정합니다. 기본값은 'm' 입니다.
- `disabled?: boolean` : 버튼의 비활성화 여부를 나타냅니다. 비활성화 시 비활성화 디자인이 나타납니다.
- `borderRadius?: string`: 버튼의 border-radius를 설정합니다. 전달하지 않을 시 기본값이 적용됩니다.
- `backgroundColor?: string`: 버튼의 배경 색상을 설정합니다. 전달하지 않을 시 기본 variant 색상이 적용됩니다.
- `color?: string`: 버튼의 글자 색상을 설정합니다. 전달하지 않을 시 기본 variant 색상이 적용됩니다.
- `borderColor?: string`: 버튼의 border 색상을 설정합니다. 전달하지 않을 시 기본 variant 색상이 적용됩니다.
- `width?: string`: 버튼의 너비를 설정합니다. 전달하지 않을 시 기본값(100%)이 적용됩니다.
- `paddingVertical?: string`: 버튼의 상하 padding을 설정합니다. 전달하지 않을 시 기본값이 적용됩니다.
- `paddingHorizontal?: string`: 버튼의 좌우 padding을 설정합니다. 전달하지 않을 시 기본값이 적용됩니다.

적용 우선순위는 custom style props > variant style(default) = size style > default style 입니다. 아래 예시를 참고하세요.

## 사용 예시

```jsx
// Basic Usage
<Button onClick={noop}>아무 디자인도 없는 버튼</Button>
<Button onClick={noop} variant="gray" backgroundColor="#ff2755">기본 variant 스타일이 적용되고 그 이후 custom 값이 적용됩니다.</Caption>
<Button onClick={noop} variant="gray" size="sm">기본 variant 스타일이 적용되고 그 이후 custom 값이 적용됩니다.</Caption>
```
