# Text

일반적인 Text 컴포넌트입니다.

p, span, div 로 시맨틱을 지정할 수 있습니다.

## 디자인

<Canvas of={TextStories.ParagraphBody1} />

## props

- `is?: 'b1' | 'b2' | 'b3'` : 나타낼 Text 컴포넌트 규격을 설정합니다.
- `as?: 'p' | 'span' | 'div'`: 원하는 시맨틱 태그를 지정합니다.
- `children: ReactNode` : Title에 나타낼 내용을 설정합니다.
- `color: keyof typeof colorAlias`: Text의 색상을 설정합니다. 기본값은 'black'입니다. color는 현재 색상의 별칭을 통해 설정됩니다. 별칭은 색상 문서를 참고하세요.

> 이외 스타일은 자유롭게 설정가능하나, 권장하지는 않습니다.

## 사용 예시

```jsx
// Basic Usage. as p tag.
<Text>빈티지 샵을 다 모았다! 빙글 Vingle</Text>
// As span tag
<Text is='b2' as='span'>빈티지 샵을 다 모았다! 빙글 Vingle</Text>
// As div tag
<Text is='b3' as='div'>빈티지 샵을 다 모았다! 빙글 Vingle</Text>
```
